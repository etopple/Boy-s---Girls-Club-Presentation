# BGCIE IT Modernization Proposal — Slide Deck

Slidev presentation for the Boys & Girls Clubs of the Inland Empire IT modernization proposal.

## Quick Start

```bash
npm install
npm run dev
```

Then open http://localhost:3030 in your browser.

## Export to PDF

```bash
npm run export
```

## Structure

- `slides.md` — Full presentation with speaker notes
- `package.json` — Slidev dependencies

## Presenter Mode

Press `P` during the presentation to enter presenter mode with speaker notes visible.
